สรุปการใช้งาน Docker กับ Flask

Docker คืออะไร?

Docker คือเครื่องมือสำหรับสร้างสภาพแวดล้อมจำลองที่เรียกว่า container ทำให้เรารันโปรแกรมได้เหมือนกันทุกเครื่อง ไม่ต้องกลัวปัญหา "เครื่องเราใช้ได้ แต่เครื่องคนอื่นใช้ไม่ได้"

การสร้าง Docker image

  1. สร้างไฟล์ Dockerfile ในโฟลเดอร์โปรเจกต์
  2. เขียนขั้นตอนใน Dockerfile เช่น ใช้ base image, ติดตั้ง library, รันแอป
  3. ใช้คำสั่ง docker build -t my-flask-app . เพื่อสร้าง image

การทดสอบรัน container

  1. ใช้คำสั่ง docker run -p 80:80 my-flask-app
  2. เปิดเว็บเบราว์เซอร์ไปที่ http://localhost เพื่อตรวจสอบว่าเว็บทำงาน

การ push image ไป Docker Hub

  1. สมัครและล็อกอิน Docker Hub
  2. ใช้คำสั่ง docker tag my-flask-app username/my-flask-app
  3. ใช้คำสั่ง docker push username/my-flask-app
  4. เข้าเว็บ Docker Hub จะเห็น image ถูกอัพโหลดไปแล้ว
