สรุปการใช้งาน Git และ GitHub

Git คืออะไร?

Git คือโปรแกรมสำหรับจัดการเวอร์ชันของโค้ด (Version Control) ใช้เก็บประวัติการแก้ไขไฟล์ ทำให้ย้อนไปดูหรือนำโค้ดเวอร์ชันเก่ากลับมาได้

GitHub คืออะไร?

GitHub คือเว็บไซต์ที่เอา Git repository ไปเก็บบนออนไลน์ ทำให้แชร์โค้ดกับคนอื่นได้ และใช้งานร่วมกันเป็นทีมได้สะดวก

ตัวอย่างขั้นตอนการใช้งาน Git + GitHub

  1. สมัครบัญชี GitHub และสร้าง Repository ใหม่
  2. ในเครื่องเราสร้างโฟลเดอร์โปรเจกต์ แล้วใช้คำสั่ง git init
  3. เพิ่มไฟล์ทั้งหมดด้วย git add .
  4. บันทึกเวอร์ชันด้วย git commit -m "first commit"
  5. ผูกกับ GitHub ด้วยคำสั่ง git remote add origin <url ของ repo>
  6. อัพโหลดโค้ดขึ้น GitHub ด้วย git push -u origin main (หรือ master แล้วแต่ชื่อ branch)
